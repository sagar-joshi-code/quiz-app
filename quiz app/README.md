🧠 Quiz App
A fun and interactive quiz app built with vanilla JavaScript that fetches real questions from an API!
🚀 Live Demo
Open index.html in your browser and start playing!
📸 Features

✅ Fetches 10 real questions from Open Trivia Database API
✅ Shows 4 shuffled answer buttons for each question
✅ Checks correct and wrong answers instantly
✅ Updates score in real time
✅ Tracks question number (Question 1 of 10)
✅ Next question button
✅ Final score screen at the end

🛠️ Technologies Used

JavaScript (Vanilla)

Open Trivia Database API
